[[common]]
== 공통 사항

API에 관계없이 아래 사항을 지켜주셔야 합니다.

=== Domain

|===
| 환경 | domain

| 운영서버
| `https://inbi-subway.kro.kr`
|===


=== Header

|===
| name | 설명

| `Authorization`
| 로그인이 필요한 API를 사용하기 위한 JWT 토큰 (Bearer ${token} 형식)
|===
